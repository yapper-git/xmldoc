# XML Syntax

## Blocks

### Headers

`h1`, `h2`, `h3`, `h4`

### Paragraph

`p` : paragraph

Alignement:
* Default is align left
* Center: `align="center"`
* Align right: `align="right"`

### Blockquote

`blockquote`

### Table

margin:auto!

* `table`
* `tr`
* `td` (`rowspan`, `colspan`)

### Lists

* `ul` : unordered list
* `ol[type]` : ordered list
    + `default` (1, 2, 3, 4 etc.)
    + `lower-alpha` (a, b, c, d etc.)
    + `upper-alpha` (A, B, C, D etc.)
    + `lower-roman` (i, ii, iii, iv etc.)
    + `upper-roman` (I, II, III, IV etc.)
* `li` : list item

## Inline

* `br` : line break
* `b` : bold
* `i` : italic
* `sup` : superscript
* `sub` : subscript
* `mark` : highlight
